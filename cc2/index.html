My Stack

HTML (HyperText Markup Language) is the standard language for creating and structuring content on the web. It defines elements like text, images, and links on a webpage.

CSS (Cascading Style Sheets) is used to style and layout HTML elements, controlling their appearance. It manages colors, fonts, spacing, and positioning.

JavaScript (JS) is a programming language used to create interactive effects and dynamic content on web pages. It enables features like form validation, animations, and data manipulation.

Go, or Golang, is an open-source programming language designed for simplicity, efficiency, and concurrency. It is often used for building scalable and high-performance applications.

SQLite is a lightweight, serverless, self-contained relational database engine. It stores data in a single file and is commonly used in mobile apps, embedded systems, and small-scale applications.

C++ (CPP) is a high-level, general-purpose programming language that supports object-oriented, procedural, and generic programming. It is widely used for system/software development and game programming.

MySQL is an open-source relational database management system (RDBMS) that uses Structured Query Language (SQL) to manage and manipulate data. It is commonly used in web applications for storing and retrieving data.
